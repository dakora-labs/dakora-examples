# Contributing

Thanks for your interest in improving these examples!

## How to Contribute

### Found an Issue?

Open an issue on [GitHub](https://github.com/dakora-labs/dakora-examples/issues) with:

- Clear description of the problem
- Steps to reproduce (if it's a bug)
- Your environment (Python version, OS)

### Want to Add an Example?

1. Fork the repository
2. Add your example to the `quickstart/` folder
3. Keep it self-contained (no external dependencies within the repo)
4. Add a clear docstring explaining what it demonstrates
5. Test that it runs successfully
6. Submit a pull request

### Improving Documentation?

Feel free to fix typos, clarify instructions, or add helpful context. Keep the style concise and practical.

## Code Style

- Follow PEP 8
- Keep examples simple and readable
- Include inline comments for complex logic
- Use type hints where helpful

## Testing Your Changes

Before submitting:

```bash
# Test that examples run
python quickstart/00_template_render.py
python quickstart/01_template_with_tracing.py
```
